# Lost & Found Portal
MERN app for reporting/searching lost and found campus items.

## Live
Frontend: https://lost-and-found-aadith.netlify.app
Backend: https://lost-and-found-portal-u13d.onrender.com

## Setup
1. Clone repo
2. `cd backend && npm i` → copy `.env.example` to `.env`, fill Mongo/JWT/Cloudinary keys → `npm run dev`
3. `cd frontend && npm i` → copy `.env.example` to `.env`, fill API URL → `npm run dev`

## Tech Stack
React, Node/Express, MongoDB (Atlas), Cloudinary, Tailwind CSS, JWT Auth
